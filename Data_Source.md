## 1. Marketaux

> **https://www.marketaux.com/**

### 핵심 특성
- **금융 뉴스 및 NLP 감성분석 전문 API** (주가/재무 데이터 미제공)
- 5,000개 이상 뉴스 소스, 80개+ 글로벌 시장, 200,000개+ 엔티티 추적

### 제공 데이터

| 카테고리 | 엔드포인트 | 설명 |
|---------|-----------|------|
| 뉴스 피드 | `GET /v1/news/all` | 글로벌 금융 뉴스 검색/필터링 |
| 유사 기사 | `GET /v1/news/similar/{uuid}` | 특정 기사와 유사한 기사 검색 |
| 감성 시계열 | `GET /v1/entity/stats/intraday` | 엔티티별 감성 추이 (분~년 단위) |
| 감성 집계 | `GET /v1/entity/stats/aggregation` | 엔티티별 감성 통계 |
| 트렌딩 | `GET /v1/entity/trending/aggregation` | 뉴스 다빈출 종목 |
| 엔티티 검색 | `GET /v1/entity/search` | 종목/자산 검색 |

### 추적 자산 유형
주식, 지수, ETF, 원자재, 뮤추얼 펀드, 선물, 통화, 암호화폐

### 고유 기능
- NLP 기반 엔티티 자동 식별 (문맥 기반)
- 기사 내 하이라이트 & 문장별 감성 점수
- 매치 스코어 (기사-종목 관련성 수치화)
- 유사 기사 자동 그룹핑
- 고급 검색 연산자 (`+`, `|`, `-`, `""`, `*`)
- 30개+ 언어 지원

### 무료 티어 제한

| 항목 | 무료 | Basic ($29) | Standard ($49) | Pro ($99) |
|------|------|-------------|----------------|-----------|
| 일일 요청 | **100회** | 2,500회 | 10,000회 | 25,000회 |
| 요청당 기사 | **3건** | 20건 | 50건 | 100건 |
| Market Stats | **불가** | 불가 | 가능 | 가능 |

### 대시보드 활용
- 뉴스 감성 시그널 모니터링
- 종목별 미디어 노출 빈도/감성 추이 시각화
- **주가/재무 데이터는 다른 소스와 병행 필수**

## 2. Finnhub

> **https://finnhub.io/**

### 핵심 특성
- **가장 포괄적인 종합 금융 데이터 플랫폼**
- 시세 + 펀더멘탈 + 대체 데이터 + 뉴스를 단일 API로 제공

### 제공 데이터

| 카테고리 | 주요 데이터 |
|---------|-----------|
| **시장 데이터** | 실시간 주가 (WebSocket), OHLCV 캔들, NBBO, 틱 데이터 |
| **기업 펀더멘탈** | 기업 프로필, 재무제표 (IS/BS/CF), 재무비율, 실적 캘린더, 임원 정보 |
| **애널리스트** | EPS/매출/EBITDA 추정치, 목표주가, 투자의견 추이 |
| **대체 데이터** | 내부자 거래, 기관 보유 (13F), **미국 의회 거래**, 로비 활동, H1B 비자, 특허, ESG |
| **뉴스/감성** | 기업 뉴스, 소셜미디어 감성, SEC 공시 감성, 어닝스콜 트랜스크립트 |
| **기술적 분석** | 기술 지표, 지지선/저항선, 패턴 인식 |
| **외환** | 통화쌍 캔들 및 환율 |
| **암호화폐** | 심볼, 캔들, 프로필 |
| **채권** | 프로필, 캔들, 수익률 곡선 |
| **ETF/펀드** | 보유 종목, 섹터/국가 비중 |
| **경제 지표** | 경제 캘린더, 국가별 역사적 데이터 |
| **공급망** | 고객/공급업체 관계 매핑, 가격 상관관계 |

### 고유 기능
- **의회 거래 데이터**: 미국 국회의원 주식 매매 활동
- **공급망 매핑**: 기업 간 관계 및 가격 상관관계
- **어닝스콜 트랜스크립트**: 실적발표 녹취록
- **SEC 공시 감성 분석**: 공시 문서 자동 감성 분석
- WebSocket 실시간 스트리밍
- Python SDK (`finnhub-python`) 공식 제공

### 지원 시장
미국 (NYSE, NASDAQ), 독일, LSE, TSX, Euronext 등 글로벌 거래소

### 무료 티어 제한
- 공통: **초당 30회** API 호출 제한
- 무료: 미국 주식 위주, 실시간은 미국 시장 일부만
- 유료: $49.99~$3,500/월 (데이터 유형별 개별 과금)

### 대시보드 활용
- 주가 차트 + 펀더멘탈 + 뉴스를 단일 소스로 구현 가능
- 의회 거래/내부자 거래 모니터링 대시보드
- 공급망 관계 시각화

## 4. FinanceDataReader

> **https://github.com/FinanceData/FinanceDataReader**

### 핵심 특성
- **한국 투자자를 위한 최적의 무료 Python 라이브러리**
- API 키 불필요, MIT 라이선스, `pip install finance-datareader`

### 제공 데이터

| 카테고리 | 주요 데이터 | 예시 |
|---------|-----------|------|
| **국내 주가** | KRX 전종목 일별 OHLCV | `fdr.DataReader('005930', '2020')` |
| **해외 주가** | NASDAQ, NYSE, SSE, SZSE, HKEX, TSE 등 | `fdr.DataReader('AAPL', '2020')` |
| **주요 지수** | 코스피, 코스닥, 다우, S&P500, 나스닥 등 | `fdr.DataReader('KS11')` |
| **환율** | USD/KRW, CNY/KRW, EUR/KRW 등 | `fdr.DataReader('USD/KRW')` |
| **암호화폐** | BTC, ETH 등 (원화/달러 기준) | `fdr.DataReader('BTC/KRW')` |
| **상품 선물** | WTI, 브렌트유, 금, 은, 구리, 천연가스 | `fdr.DataReader('GC=F')` |
| **미국 국채** | 5/10/30년 수익률 | `fdr.DataReader('US10YT')` |
| **종목 리스팅** | KRX/KOSPI/KOSDAQ/KONEX + 시가총액순 | `fdr.StockListing('KRX')` |
| **ETF 리스팅** | 한국 ETF 전종목 | `fdr.StockListing('ETF/KR')` |
| **재무제표** | K-IFRS/K-GAAP 별도/연결 (연간/분기) | `fdr.SnapDataReader('NAVER/FINSTATE/005930')` |
| **KRX 지수 구성종목** | 코스피200, ESG 지수 등 | `fdr.SnapDataReader('KRX/INDEX/STOCK/1028')` |
| **FRED 경제지표** | M2 통화량, 장단기금리차 등 | `fdr.DataReader('FRED:M2')` |
| **ECOS 한은 통계** | 기준금리, 100대 경제지표 | `fdr.DataReader('ECOS:한국은행기준금리')` |
| **상장폐지/관리종목** | 과거 가격 포함 | `fdr.StockListing('KRX-DELISTING')` |

### 지원 거래소

| 거래소 | 코드 | 종목수 |
|-------|------|--------|
| 한국거래소 (통합) | KRX | 2,663+ |
| 코스피 | KOSPI | 940+ |
| 코스닥 | KOSDAQ | 1,597+ |
| 나스닥 | NASDAQ | 4,000+ |
| 뉴욕증권거래소 | NYSE | 3,000+ |
| 상해증권거래소 | SSE | 1,000+ |
| 선전증권거래소 | SZSE | 1,000+ |
| 홍콩증권거래소 | HKEX | 2,500+ |
| 도쿄증권거래소 | TSE | 3,900+ |

### 한국시장 특화 기능 (핵심 차별점)
1. **KRX 종목코드 자동 인식** (6자리 숫자 → 한국 종목, 2024년 영문자 코드 대응)
2. **시가총액순 종목 리스팅** (현재가, PER, PBR, ROE, 외국인비율 등 34개 필드)
3. **K-IFRS/K-GAAP 재무제표** (별도/연결, 연간/분기)
4. **상장폐지/관리종목** 과거 데이터 조회
5. **ECOS 한국은행 통계** 한글 키워드 직접 조회
6. **원화 기반** 암호화폐/환율 직접 지원

### 제한사항
- **일별 데이터만** (분봉/틱/실시간 미지원)
- 웹 크롤링 기반이라 소스 사이트 변경 시 영향 가능
- 뉴스/감성분석/기술지표 미제공
- 해외 ETF 리스팅 잠정 중단

### 대시보드 활용
- **국내 주식 대시보드의 핵심 데이터 소스**
- 종목 스크리닝 (시가총액, PER, PBR, ROE 등)
- 재무제표 분석
- 환율/원자재/경제지표 모니터링
